四、安装命令
--------------------------------------------------------------------------------

【使用 uv（推荐）】
uv sync
uv run src/main.py
uv run python backend/app.py

【使用 pip】
pip install -e .
pip install fastapi uvicorn
python run.py
python backend/app.py

【启动前端】
cd frontend
python -m http.server 8080

五、服务地址
--------------------------------------------------------------------------------
| 服务 | 地址 |
|------|------|
| CLI | 命令行直接交互 |
| 后端 API | http://localhost:8001 |
| 前端界面 | http://localhost:8080 |

六、快速开始
--------------------------------------------------------------------------------
1. git clone https://gitee.com/chain-engine/x-langchain.git
2. cd x-langchain
3. cp .env.example .env
4. 编辑 .env 配置 API Key
5. python run.py (CLI模式) 或 python backend/app.py (Web模式)
